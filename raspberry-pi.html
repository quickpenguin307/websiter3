Raspberry Pi Starter Project

DiscereMinds | Empowering STEM Education

Overview

In this Raspberry Pi starter project, you'll learn how to set up and use your Raspberry Pi to blink an LED. This is a great beginner-level project that introduces you to programming in Python and working with the GPIO pins on the Raspberry Pi.

Components Required

  • Raspberry Pi (any model with GPIO pins, e.g., Raspberry Pi 4)
  • LED
  • 220-ohm resistor
  • Jumper wires
  • Breadboard
  • Raspberry Pi power supply
  • Internet connection to set up Raspberry Pi (optional)
  • Python IDE (pre-installed on Raspberry Pi OS)

Setting Up the Circuit

Here are the steps to connect the components to your Raspberry Pi:

  1. Insert the LED into the breadboard with the longer leg (positive) connected to a free row and the shorter leg (negative) connected to another row.
  2. Place a 220-ohm resistor between the shorter leg (negative) of the LED and the ground (GND) row on the breadboard.
  3. Connect a jumper wire from the positive leg of the LED to GPIO pin 18 on the Raspberry Pi (this is one of the general-purpose input/output pins).
  4. Connect another jumper wire from the ground (GND) row of the breadboard to one of the GND pins on the Raspberry Pi.

Programming the Raspberry Pi

Follow these steps to write and run the Python program to blink the LED:

  1. On your Raspberry Pi, open the terminal and type sudo nano blink.py to create a new Python script.
  2. Enter the following Python code to blink the LED:
import RPi.GPIO as GPIO
import time

# Set up the GPIO pin
GPIO.setmode(GPIO.BCM)
GPIO.setup(18, GPIO.OUT)

# Blink the LED
while True:
    GPIO.output(18, GPIO.HIGH)  # Turn LED on
    time.sleep(1)                # Wait for 1 second
    GPIO.output(18, GPIO.LOW)   # Turn LED off
    time.sleep(1)                # Wait for 1 second

Save the file by pressing CTRL + X, then Y to confirm, and Enter to exit.

Run the script by typing python3 blink.py in the terminal. You should see the LED blink on and off every second.

Observations

As the program runs, the LED will blink on and off at a 1-second interval. This project helps you understand basic Python programming and how to control hardware using GPIO pins on the Raspberry Pi.

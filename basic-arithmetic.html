Understanding Basic Arithmetic Operations

In this lesson, we will break down how to perform basic arithmetic operations: addition, subtraction, multiplication, and division. By the end of this page, you'll know how to solve problems using these operations step-by-step.

1. Addition

Addition is the process of combining two or more numbers to get a total. To perform addition:

  1. Start with the first number.
  2. Add the second number to the first number.
  3. If the sum is more than 9, carry over any tens to the next place value.

For example: 23 + 17

  • Start with 23.
  • Now add 17.
  • In the ones column: 3 + 7 = 10, so write 0 and carry over 1.
  • In the tens column: 2 + 1 + 1 (carried over) = 4.

Therefore, 23 + 17 = 40.

2. Subtraction

Subtraction is the process of taking one number away from another. To perform subtraction:

  1. Start with the number you're subtracting from (the minuend).
  2. Take away the second number (the subtrahend) from it.
  3. If the top number is smaller than the bottom number, borrow from the next column.

For example: 45 - 19

  • Start with 45.
  • Now subtract 19.
  • In the ones column: 5 - 9. Since 5 is less than 9, we borrow 1 from the tens column (so 4 becomes 3) and make the 5 a 15.
  • Now subtract: 15 - 9 = 6.
  • In the tens column: 3 - 1 = 2.

Therefore, 45 - 19 = 26.

3. Multiplication

Multiplication is the process of adding the same number several times. To perform multiplication:

  1. Start with the first number (the multiplicand).
  2. Multiply it by the second number (the multiplier).
  3. To do this, add the multiplicand to itself as many times as the multiplier indicates.

For example: 6 × 7

  • Start with 6.
  • Add 6 to itself 7 times: 6 + 6 + 6 + 6 + 6 + 6 + 6.
  • That's 42.

Therefore, 6 × 7 = 42.

4. Division

Division is the process of splitting a number into equal parts. To perform division:

  1. Start with the dividend (the number being divided).
  2. Divide it by the divisor (the number you are dividing by).
  3. If necessary, write the remainder as a fraction or decimal.

For example: 56 ÷ 8

  • Start with 56.
  • Now divide by 8. 56 ÷ 8 = 7, with no remainder.

Therefore, 56 ÷ 8 = 7.

Word Problems: How to Solve Them

Word problems are a way to apply arithmetic to real-life situations. Here's how to solve them:

  1. Read the problem carefully to understand what is being asked.
  2. Identify the operation needed (addition, subtraction, multiplication, or division).
  3. Translate the problem into a mathematical expression.
  4. Solve the problem step-by-step.

Example Word Problem: If you have 3 apples and you buy 5 more, how many apples do you have in total?

  • Operation: Addition.
  • Mathematical expression: 3 + 5.
  • Solution: 3 + 5 = 8 apples.

Practice Problems

Try solving these problems on your own:

  • What is 58 + 22?
  • What is 144 ÷ 12?
  • What is 5/8 + 3/8?
  • What is 30% of 80?

Remember, practice is the key to mastering arithmetic. Work through problems step-by-step, and you’ll get better over time.

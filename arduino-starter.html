Arduino Starter Project

Overview

In this basic Arduino starter project, you will learn how to light up an LED using your Arduino board. This simple project is perfect for beginners and helps you understand the fundamentals of working with Arduino and electronic components.

Components Required

  • Arduino board (e.g., Arduino Uno)
  • LED
  • 220-ohm resistor
  • Breadboard and jumper wires
  • USB cable to connect Arduino to your computer

Setting Up the Circuit

Follow these steps to connect the components:

  1. Place the LED on the breadboard, ensuring the longer leg (positive) is connected to a row on the breadboard.
  2. Connect a 220-ohm resistor from the shorter leg (negative) of the LED to the ground (GND) row on the breadboard.
  3. Connect a jumper wire from the positive leg of the LED to digital pin 13 on the Arduino board.
  4. Connect another jumper wire from the ground row of the breadboard to a GND pin on the Arduino board.

Programming the Arduino

Now, you will program your Arduino to light up the LED. Follow these steps:

  1. Open the Arduino IDE on your computer.
  2. In the IDE, go to File > New to create a new sketch.
  3. Write the following code into the sketch:
void setup() {
  pinMode(13, OUTPUT); // Set pin 13 as an output
}

void loop() {
  digitalWrite(13, HIGH); // Turn the LED on
  delay(1000); // Wait for 1 second
  digitalWrite(13, LOW); // Turn the LED off
  delay(1000); // Wait for 1 second
}

After writing the code, click the upload button in the Arduino IDE to upload the code to your Arduino board.

Observations

When the program runs, the LED should blink on and off every second. This project helps you understand basic Arduino programming and how to control hardware with code.

Light Sensor Experiment

DiscereMinds | Empowering STEM Education

Overview

In this project, you'll build a simple light sensor using an Arduino and a photoresistor (LDR). The sensor will detect ambient light levels and output the readings to the Serial Monitor.

Components Required

  • Arduino Uno (or similar)
  • Photoresistor (LDR)
  • 10k ohm resistor
  • Breadboard
  • Jumper wires

Circuit Diagram

Connect the photoresistor in series with a 10k ohm resistor to form a voltage divider. Connect the junction between the photoresistor and resistor to the Arduino's analog input pin A0.

Arduino Code

Upload the following code to your Arduino board using the Arduino IDE:


void setup() {
  Serial.begin(9600);  // Initialize serial communication at 9600 bps
}

void loop() {
  int sensorValue = analogRead(A0);  // Read the input on analog pin A0
  Serial.print("Light Sensor Value: ");
  Serial.println(sensorValue);       // Print the value to the Serial Monitor
  delay(500);                        // Wait for 500 milliseconds before the next reading
}

Instructions

  1. Gather all the required components.
  2. Assemble the circuit as described in the circuit diagram.
  3. Connect your Arduino to your computer and open the Arduino IDE.
  4. Upload the provided code to your Arduino.
  5. Open the Serial Monitor (set to 9600 baud) to view the sensor readings in real time.
  6. Experiment by changing the light conditions around the sensor and observe the changes in the sensor value.
